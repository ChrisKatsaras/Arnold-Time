# ARNOLD TIME!:fire:
A simple online shooter to play with your friends

## Controls

### Move:
<kbd>W</kbd><kbd>A</kbd><kbd>S</kbd><kbd>D</kbd> to move   

### Rotate:
Use mouse to rotate character

### Shoot:  
Click to shoot  

### Shield
<kbd>Spacebar</kbd> to activate shield
